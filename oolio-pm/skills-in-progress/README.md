# Skills in progress

Skills being designed or trialled. Nothing in this folder is shipped: the plugin loader reads only `oolio-pm/skills/`, so work here can change or die without ceremony. Each candidate gets its own folder with a `HANDOFF.md` (the design brief a fresh session can build from) until it has a working SKILL.md.

Lifecycle rules in `../references/skill-standard.md`: promotion to `skills/` is a CHANGELOG entry and a status of **New**; abandonment moves the folder to `_archive/` with a note.
