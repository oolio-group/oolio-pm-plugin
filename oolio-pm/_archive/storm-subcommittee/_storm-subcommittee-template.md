# STORM Subcommittee lens template

Copy this file into `storm-subcommittee/` to add a research lens to the panel. Fill every section. Empty sections are not allowed. Keep it to one file. British English only. No em dashes. No buzzwords.

These are research roles, not user personas and not opinionated executive seats. Each one is a part of the STORM and Co-STORM method, written as a function. Do not attach a real person's name or invented quotes. The challenge questions are what the role asks while researching.

This template follows the same shape as the Leadership Subcommittee template, so the library reads consistently.

Replace the placeholder text in `[brackets]` and remove the bracket markers when done.
