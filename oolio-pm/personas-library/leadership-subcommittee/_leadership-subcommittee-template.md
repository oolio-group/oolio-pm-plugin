# Leadership Subcommittee lens template

Copy this file into `leadership-subcommittee/` to add an executive or commercial lens to the panel. Fill every section. Empty sections are not allowed. Keep it to one file. British English only. No em dashes. No buzzwords.

These are lenses, not user personas. A user persona (`../uat-panel/`) is a real-feeling hospitality operator we build for. A Leadership Subcommittee lens is an executive or commercial point of view we test a decision against. Write it as a function (CPO, CFO, CRO), never as a named individual, and never with invented quotes.

This template blends the council structure (role, primary concern, what it protects, challenge questions, review output) with the house style used across the library, including the "where this lens clashes" section that keeps the panel honest.

Replace the placeholder text in `[brackets]` and remove the bracket markers when done.
